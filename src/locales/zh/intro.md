## 在你开始之前...

首先，虽然我们为了中文用户的方便提供了中文的表单，但在填写时请**尽量使用英文**。WuxUI 社区不只有中国人。如果你想让尽可能多的人能够看懂你的 issue，就请尽量用英文。如果担心表达不清楚，可以同时提供中文内容。

其次，WuxUI 的 issue 列表只接受 bug 报告或是新功能请求 (feature requests)。这意味着**我们不接受用法问题**。如果你开的 issue 不符合规定，它将会被**立刻关闭**。[为什么要这么严格？](#intro-modal)

对于使用中遇到的问题，请使用以下资源：

- 仔细阅读 <a href="https://www.wuxui.com/#/quickstart" target="_blank">使用文档</a> 和组件文档
- 提问前确保你在 <a href="https://www.wuxui.com/#/faq" target="_blank">常见问题</a> 和 <a href="https://www.wuxui.com/#/changelog" target="_blank">更新日志</a> 中搜索过

最后，在开 issue 前，可以先搜索一下以往的旧 issue - 你遇到的问题可能已经有人提了，也可能已经在最新版本中被修正。注意：如果你发现一个已经关闭的旧 issue 在最新版本中仍然存在，请不要在旧 issue 下面留言，而应该用下面的表单开一个新的 issue。
